# 6장 메모: 검색으로 목록 필터링하기
## 장 목표

- `TextInput` 검색어에 따라 `FlatList` 결과가 실시간으로 바뀌는 흐름을 익힌다.
- 원본 목록 상태와 화면에 보여 줄 필터 결과를 구분해 이해한다.
- 이름과 역할 텍스트를 기준으로 간단한 검색 기능을 구현한다.
- chapter 05의 목록 예제를 검색 가능한 목록 화면으로 확장한다.

## 구성 초안

1. 왜 목록 필터링이 필요한가
2. 검색어 상태 만들기
3. 원본 배열에서 조건에 맞는 항목만 고르기
4. 필터 결과를 FlatList에 연결하기
5. 검색 결과가 없을 때 처리하기
6. 실행 결과 확인
7. 정리

## 실습 메모

- chapter 06은 chapter 05의 목록 구조를 유지하되, 추가 기능 대신 검색에 집중한다.
- 이번 장의 핵심은 `검색어 상태 -> 필터 결과 계산 -> FlatList 재렌더링` 흐름이다.
- 서버 검색이 아니라 로컬 배열 필터링으로 기본기를 먼저 익힌다.
