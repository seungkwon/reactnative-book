# 10장 메모: 선택한 이미지를 프로필 화면에 적용하기
## 장 목표

- 9장에서 선택한 이미지 결과를 실제 프로필 카드 UI에 연결한다.
- 이미지 URI를 상태로 관리하고, 프로필 사진 자리와 설명 문구에 반영한다.
- 네이티브 기능 결과가 앱 화면의 일부가 되는 흐름을 이해한다.

## 구성 초안

1. 왜 이미지 선택 결과를 화면에 적용해야 하는가
2. 프로필 카드 구조 만들기
3. 갤러리/카메라 선택 로직 재사용하기
4. 선택한 이미지를 아바타에 반영하기
5. 상태 문구와 프로필 설명 연결하기
6. 실행 결과 확인
7. 정리
