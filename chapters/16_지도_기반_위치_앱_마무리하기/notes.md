# 16장 메모: 지도 기반 위치 앱 마무리하기
## 장 목표

- 현재 위치 이동과 주소 검색을 하나의 지도 화면으로 통합한다.
- 선택된 위치 정보를 카드 형태로 정리해 실제 앱 같은 흐름을 만든다.
- 지도 기능 장들을 하나의 위치 기반 앱 구조로 마무리한다.

## 구성 초안

1. 위치 앱 흐름 통합하기
2. 현재 위치와 주소 검색 기능 결합
3. 선택 위치 카드 구성
4. 지도 중심 이동과 마커 갱신
5. 결과 화면 확인
6. 정리

## 학습 메모

- 이번 장은 새 라이브러리보다 기존 지도 기능들을 어떻게 하나의 앱 흐름으로 묶는지가 핵심이다.
- 현재 위치와 주소 검색은 결국 같은 지도 상태를 갱신하는 두 개의 입력 경로로 볼 수 있다.
- 사용자 입장에서는 좌표보다 선택된 위치 카드와 버튼 흐름이 훨씬 이해하기 쉽다.
