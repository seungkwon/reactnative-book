# 8장 메모: 목록에서 상세 화면으로 이동하기
## 장 목표

- 목록에서 선택한 항목의 상세 정보를 별도 영역에 표시하는 흐름을 익힌다.
- 선택 상태를 이용해 화면의 일부가 바뀌는 패턴을 이해한다.
- 목록 화면과 상세 화면 역할을 하나의 예제 안에서 연결해 본다.
- chapter 07까지 만든 목록 예제를 선택 기반 상세 보기로 마무리한다.

## 구성 초안

1. 왜 상세 화면 이동이 필요한가
2. 선택 상태 만들기
3. 목록 항목 클릭으로 선택값 바꾸기
4. 선택된 프로필 상세 카드 렌더링하기
5. 실행 결과 확인
6. 정리
