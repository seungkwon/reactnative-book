# 5장 메모: 입력으로 목록 항목 추가하기
## 장 목표

- `TextInput` 입력값을 받아 `FlatList` 목록에 새 항목을 추가한다.
- 배열 상태를 확장하는 패턴을 익힌다.
- 사용자의 입력과 버튼 클릭이 목록 구조를 직접 바꾸는 흐름을 이해한다.
- chapter 04의 읽기 전용 목록을 쓰기 가능한 목록으로 확장한다.

## 구성 초안

1. 왜 목록 추가 기능이 필요한가
2. 입력 상태와 목록 상태 함께 다루기
3. 새 프로필 데이터 만들기
4. 버튼 클릭으로 배열 맨 앞에 항목 추가하기
5. 실행 결과 확인
6. 정리
